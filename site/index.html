Installation

Recommended and the fastest way to get started is to use CDN
#

CDN

#

unpkg

unpkg|version , hits etc
href="https://unpkg.com/@aman-atg/normal.css"
#

jsDelivr

jsdelivr|version , hits etc
https://cdn.jsdelivr.net/npm/@aman-atg/normal.css

NPM

$ npm i @aman-atg/normal.css

CDN

<link rel="stylesheet" href="https://unpkg.com/@aman-atg/normal.css" />
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@aman-atg/normal.css" />

CSS/SCSS

@import url("https://unpkg.com/@aman-atg/normal.css");

JS

@import "@aman-atg/normal.css";
require "@aman-atg/normal.css";
require "path/to/local/normal.min.css";
